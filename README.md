# freeks-coach



## Stack

- Runtime: Bun
- Test: Vitest
- Lint/format: Biome
- Type-check: TypeScript

## Aan de slag

```bash
bun install
bun test
bun run typecheck
bun run check
```

## LabTech-werkstroom

Deze repo gebruikt [LabTech](https://github.com/) — zie `CLAUDE.md` en `.claude/labtech.json` voor tier-info.

- Experiment? `/start-experiment <naam>` (in een Claude-sessie).
- Klaar? `/promote-to-main`.
